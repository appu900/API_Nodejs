# 📝 Task Manager API

A simple Task CRUD (Create, Read, Update, Delete) RESTful API built with Node.js, Express, and MongoDB.

---

## 🚀 Features

- Create a new task
- View all tasks
- Mark a task as completed
- Delete a task

---

## 🛠️ Tech Stack

- **Backend**: Node.js, Express
- **Database**: MongoDB (with Mongoose)
- **HTTP Client**: Fetch / Postman

---
